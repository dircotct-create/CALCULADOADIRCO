Fórmulas usadas

Consumo tinta = tinta_ml * costo_tinta_ml
Labor por unidad = (tiempo_min / 60) * costo_mano_obra_hora
Energia por unidad = (tiempo_min / 60) * costo_energia_hora
Coste base = base + papel + tinta + labor + energia + otros
